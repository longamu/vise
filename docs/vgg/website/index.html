VGG Image Search Engine (VISE)

Abhishek Dutta, Relja Arandjelovic, and Andrew Zisserman

Overview

VGG Image Search Engine (VISE) is an open source project developed at the Visual Geometry Group and released under the BSD-2 clause license. This standalone application can be used to make a large collection of images searchable by using image regions as query.

VISE builds on the C++ codebase developed by Relja Arandjelovic during his DPhil / Postdoc at the Visual Geometry Group, Department of Engineering Science, University of Oxford.
